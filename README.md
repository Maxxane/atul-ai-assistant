# Atul Rao – AI Career Assistant

## Overview
This project is a **personal AI assistant** built to help recruiters and hiring managers learn about Atul Rao. It uses **OpenAI’s GPT models** to answer questions about my **professional background, skills, projects, and career goals**, grounded in verified information.  

It demonstrates my **applied AI engineering skills**, including:
- Prompt engineering  
- API design with **FastAPI**  
- Knowledge-grounded AI responses  
- Professional-grade project packaging

---

## Features
- Answer recruiter questions based on **verified professional profile**  
- Friendly, professional, and concise responses  
- Safe and non-hallucinating AI output  
- Easily extendable to include **new projects, skills, and learning areas**

---

## Tech Stack
- **Backend:** Python, FastAPI  
- **AI/ML:** OpenAI GPT (gpt-4o-mini)  
- **Deployment:** Local/Cloud-ready  
- **Other:** `.env` for API key management
